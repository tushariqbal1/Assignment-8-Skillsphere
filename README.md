# SkillSphere - Online Learning Platform

![SkillSphere](https://via.placeholder.com/800x400/6366f1/ffffff?text=SkillSphere)

A modern, beautiful, and fully responsive online learning platform where users can explore courses, watch lessons, and enroll in skill-based programs.

## 🚀 Live Demo
**Live URL:** `assignment-8-skillsphere-seven.vercel.app` *(Update after deployment)*

## ✨ Features

- **Modern & Responsive Design** – Fully responsive on mobile, tablet, and desktop
- **User Authentication** – Email/Password + Google OAuth Login
- **Course Browsing** – Search, category & level filters
- **Course Details** – Detailed view with curriculum and description
- **Protected Routes** – Profile page only accessible after login
- **Profile Management** – Update personal information
- **Beautiful UI/UX** – Built with Tailwind CSS + DaisyUI
- **Smooth Interactions** – Hero slider with Swiper.js
- **Mobile Friendly** – Hamburger menu with smooth navigation

## 🛠️ Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS + DaisyUI
- **Authentication:** BetterAuth
- **Carousel:** Swiper.js
- **Icons:** Lucide React
- **Language:** JavaScript (JSX)
